# NarxozEat - University Food Delivery Platform

## Project Overview
This is an MVP developed for the **ISC-1** project at **Narxoz University**. 
The goal of the project is to reduce queues in campus cafeterias by providing a pre-order system.

## Tech Stack
- **Backend:** Node.js with Express
- **Frontend:** HTML5/CSS3 (Narxoz Corporate Style)

## How to Run
1. Install Node.js
2. Run `npm install`
3. Start the server with `node index.js`
4. Open `http://localhost:3000`
